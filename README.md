# UNSDG Goals classifier tool for your repository

A responsive website for categorizing the open source projects into different UN SDG's goals that closely align with their repository by reading their directory.

# Instructions to run in your machine

1. Clone the repository to your local machine using `git clone https://github.com/gw-ospo/UNSDG-Goals-classifier-tool.git`.
2. To setup the backend project, go to backend folder and follow the instructions:
   a. Install all the dependancies by using the command `pip install -r requirements.txt`
   b. Run the program by command `python app.py`.
   c. The server should run at `https://127.0.0.1:5000`

3. To setup the frontend project, go to frontend folder and follow the instructions:
   a. Install all the dependancies by using the command `npm install`
   b. Run the program by command `npm run dev`.
   c. The server should run at `https://127.0.0.1:3000`

4. Enjoy!
